### Social-Network WebApp

#### Tech Stack
- Redux
- Tailwind CSS
- React JS
- MockBee BackEnd

#### Features
- Login, SignUp, Logout
- Create, Update, Delete Post
- Like, Dislike Post
- Add, Remove Bookmark
- Create, Update, Delete Comment
- Sort by Like
- Profile Update
- Responsive site
